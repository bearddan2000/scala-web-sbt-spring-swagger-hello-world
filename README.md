# scala-web-sbt-spring-swagger-hello-world

## Description
A POC for swagger in springframework.

## Tech stack
- scala
- sbt

## Docker stack
- openjdk:8-jre-alpine

## To run
`sudo ./install.sh -u`
- API http://localhost/hello
- Raw JSON http://localhost/v2/api-docs
- Swagger http://localhost/swagger-ui.html

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
